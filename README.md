# yabai-skhd-config

My macOS tiling window manager config. Focus (Cmd+arrows), swap (Shift+Cmd+arrows), resize (Ctrl+Cmd+arrows), spaces (Alt+1..9), float/fullscreen/layout controls, display hotplug auto-recovery. 31 yabai + 1 iTerm2 shortcut.

## Install

```bash
ln -sf ~/yabai-skhd-config/yabai/.yabairc ~/.yabairc
ln -sf ~/yabai-skhd-config/skhd/.skhdrc ~/.skhdrc
```
